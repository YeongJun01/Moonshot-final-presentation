프로젝트 내용

본 프로젝트는 노션과 유사한 프로젝트 일정 관리 서비스를 구현하는 것을 목표로 하며, 요규사항 분석을 통해 프로젝트 및 할 일 관리 기능을 설계하고, 구글 로그인과 캘린더 연동을 포함한 핵심 기능을 구현했습니다.

개발 환경 및 사용 스택

  • Backend: Node.js, Express
  • Database: PostgreSQL
  • Deployment: Render(서버), Netlify(발표자료)
  • API 문서화: Swagger
  • 협업 도구: Discord, Github
